Requerimientos del Sistema de Gestión de Turnos y Pagos

 Actores
- Empleado
- Administrador

---

Tabla de Requerimientos

| Actor        | Requerimiento Funcional                                      | Requerimiento No Funcional                                      |
|--------------|---------------------------------------------------------------|------------------------------------------------------------------|
| Empleado     | Registrar inicio y fin de turno                               | El sistema debe responder en menos de 3 segundos                 |
| Empleado     | Consultar horas trabajadas                                    | La información debe mostrarse de forma clara                     |
| Empleado     | Consultar pagos realizados                                    | La plataforma debe ser accesible desde celular y PC              |
| Empleado     | Descargar comprobante de pago                                 | Los archivos deben descargarse en menos de 5 segundos            |
| Administrador| Registrar empleados                                            | El acceso debe estar protegido con contraseña                    |
| Administrador| Revisar y aprobar turnos                                       | Solo usuarios autorizados pueden aprobar pagos                   |
| Administrador| Generar reportes de pagos                                      | Los reportes deben generarse sin errores                          |
| Administrador| Modificar datos de empleados                                   | El sistema debe guardar respaldo automático                      |
| Administrador| Calcular pagos automáticamente                                 | El cálculo debe ser preciso y sin fallos                          |
| Administrador| Enviar notificaciones de pago                                  | Las notificaciones deben enviarse en tiempo real                 |
